Hi, I'm a grandma and a developer. Clean and elegant code is my passion.

A little bit about me.

I'm a young grandma who started coding with the Commadore 64 way back in the world of tape recorders. If my brother and sister wanted to play a game, we had to copy code from magazines. Sometimes the code wasn't correct, and since we didn't have the following issue, I had to debug them myself. It was a great challenge! When I started working, I was always looking for better ways to use the technology available. I've coded macros in WordBasic, (not VisualBasic, WordBasic) and

loved the challenge of making things work the way I wanted them to. In my last position, I created an Excel spreadsheet that would look up and display information based on the users input. I wanted to build a more sophisticated interface, but didn't have the tools for the job. This peaked my curiousity and I couldn't seem to let it go. The world of web development is just opening for me, but the possibilities are endless and facinating.
